## 📋 About homework-grader

homework-grader is a tool that uses AI to score homework based on clear grading rules. It is designed to help teachers and educators quickly review many student submissions. The system works by applying a set of instructions called a rubric to each piece of homework. It uses careful reasoning and checks to reduce bias. The application also follows a quality cycle to improve grading accuracy over time.

This tool runs on Windows and exports results in Excel format for easy tracking. You do not need any special technical skills to use it.

## 🔄 How homework-grader Works

This tool uses artificial intelligence based on Claude, a large language model (LLM). It reads each answer and scores it against your rubric while thinking step-by-step. This method helps explain how it arrived at each score.

The system also uses bias checks to avoid unfair scoring based on student background or style. Finally, it applies a quality cycle called PDCA (Plan-Do-Check-Act) to consistently improve grading results.
